Question One


Given a search string S and a subject string X, write a function which will return the minimum window in X which contains all the characters in S.

Example:

  • X = “ADOBECODEBANC”
  • S = “ABC”

Question Two


equilibrium indices

Write a function that, given a zero-indexed array A consisting of N integers, returns any of its equilibrium indices. The function should return -1 if no equilibrium index exists.

Assume that:

N is an integer within the range [0..10,000,000];

each element of array A is an integer within the range [-2,147,483,648..2,147,483,647].

Complexity:

  • Expected worst-case time complexity is O(N);
  • Expected worst-case space complexity is O(N), beyond input storage (not counting the storage required for input arguments).
  • Elements of input arrays can be modified.

Question Three


distinct values

Write a function that, given a zero-indexed array A consisting of N integers, returns the number of distinct values in array A. Assume that: N is an integer within the range [0..100,000]; each element of array A is an integer within the range [-1,000,000..1,000,000].

For example, given array A consisting of six elements such that:
A[0] = 2 A[1] = 1 A[2] = 1
and
A[3] = 2 A[4] = 3 A[5] = 1

The function should return 3, because there are 3 distinct values appearing in array A, namely 1, 2 and 3.

Complexity:

  • expected worst-case time complexity is O(N*log(N));
  • expected worst-case space complexity is O(1), beyond input storage (not counting the storage required for input arguments).
  • Elements of input arrays can be modified.
